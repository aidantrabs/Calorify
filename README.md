### Technologies :computer:

* [Python] - Used for the backend
* [Flask] - Framework utilized
* [HTML] - Frontend
* [CSS] - To make it look pretty

### Installation :wrench:

Calorify requires [Flask] to run.

Install the dependencies.

```sh
$ pip install Flask
$ pip install nutritionix
```
